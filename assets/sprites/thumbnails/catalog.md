## Reading an icon in Python

```python
def get_icon(sheet: pygame.Surface, icon_id: int) -> pygame.Surface:
    col = icon_id % 10
    row = icon_id // 10
    x = col * 50 + 2
    y = row * 39
    return sheet.subsurface(pygame.Rect(x, y, 46, 38))
```
